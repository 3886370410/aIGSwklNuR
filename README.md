# 项目简介

## 核心功能
- 基于YOLOv8算法实现特种车辆检测，包含5类特种车辆识别：卡车、消防车、自卸车、警车和工程车辆
- 支持26000张标注数据集的完整训练模型
- 提供多种检测方式：单张图片、图片文件夹、视频文件和本地摄像头实时检测
- 界面功能包括：实时显示目标位置、目标总数、置信度等信息，支持自定义置信度和IOU阈值
- 支持选择不同检测模型，可选择性显示标签和原图
- 提供批量检测功能，界面内可直接查看所有检测结果
- 支持检测结果的保存功能

## 适用场景
- 交通监控系统中的特种车辆识别与管理
- 安防领域中的异常车辆检测
- 工地现场的工程车辆安全管理
- 应急响应系统中的消防车快速识别
- 执法系统中的警车识别与追踪

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python 3.9、OpenCV-Python、PyQt5、Ultralytics和PyTorch 1.9等技术栈
- 提供完整的UI界面源文件，包括图标和界面设计
- 已配置成功环境，可直接运行使用
